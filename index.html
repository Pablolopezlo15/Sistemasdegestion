SISTEMAS DE GESTIÓN DE INFORMACIÓN

ERP

¿Qué es?

ERP son las siglas de "Enterprise Resource Planning", que en español se traduce como "Planificación de Recursos Empresariales". Es un sistema integrado de gestión empresarial que permite a las organizaciones planificar, gestionar y controlar sus recursos y procesos empresariales de manera más eficiente. Un ERP normalmente incluye módulos para la gestión de finanzas, contabilidad, compras, inventario, producción, recursos humanos entre otros.

CRM

¿Qué es?

CRM son las siglas de "Customer Relationship Management" que en español se traduce como "Gestión de la relación con el cliente". Un CRM es un sistema o software que permite a las empresas gestionar y mantener relaciones con sus clientes. Un CRM normalmente incluye herramientas para la gestión de ventas, marketing, servicio al cliente y análisis de datos.

BI

¿Qué es?

BI son las siglas de "Business Intelligence", que en español se traduce como "Inteligencia de negocios". Son un conjunto de herramientas y técnicas que permiten a las empresas analizar y entender grandes cantidades de datos que le serán de gran ayuda a la empresa para tomar decisiones.

SISTEMAS DE REFUSIÓN O SINDICACIÓN DE CONTENIDOS

RSS

¿Qué es?

RSS son las siglas de "Really Simple Syndication" que se traduce como "Sindecación Realmente Simple". Es una tecnología que permite a los usuarios recibir actualizaciones de contenido web de forma automática y en tiempo real, sin necesidad de estar visitando constantemente las paginas web de la que proviene el software que este ha adquirido.

ATOM

¿Qué es?

Atom es un formato de sindicación de contenidos web similar a RSS, que se utiliza para publicar y distribuir contenidos en internet. Se desarrolló como una alternativa abierta y estandarizada al formato RSS.
